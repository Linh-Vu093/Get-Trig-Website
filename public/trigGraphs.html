Trigonometric Graphs and Functions

Glossary of Terms

Asymptote: A line that a graph approaches but never touches.
Dilation: A transformation that changes the size of a figure without altering its shape.
Period: The horizontal length of one complete cycle of a trigonometric function.
Amplitude: The maximum distance from the midline (or equilibrium position) to the peak or trough of a trigonometric graph.
Domain: The set of all possible input values (usually x-values) for which a trigonometric function is defined.
Range of a function: The set of all possible output values of a function.
Horizontal line test: A test to determine if a function is one-to-one by checking if any horizontal line intersects the graph more than once.
Interval notation: A way to describe a range of values, such as \( (a, b) \) for all values between \( a \) and \( b \), not including \( a \) and \( b \).
Limit: The value that a function approaches as the input approaches a certain point.
Even function: A function that is symmetric about the y-axis, meaning \( f(x) = f(-x) \).
Odd function: A function that is symmetric about the origin, meaning \( f(-x) = -f(x) \).
