Trigonometric Identities and Functions

Trigonometric identities are equations involving trigonometric functions that are true for all values of the variables where both sides are defined. Some fundamental identities include:

  • Pythagorean Identity: sin²θ + cos²θ = 1
  • Reciprocal Identities: sinθ = 1/cscθ, cosθ = 1/secθ, tanθ = 1/cotθ
  • Quotient Identities: tanθ = sinθ/cosθ, cotθ = cosθ/sinθ
  • Co-Function Identities: sin(90°−θ) = cosθ, cos(90°−θ) = sinθ

Understanding these identities is essential for solving trigonometric equations and simplifying expressions.
